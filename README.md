# 💻 Hemanth Kumar Portfolio

A highly interactive personal portfolio built from scratch using HTML, CSS, and JavaScript.

## ✨ Highlights
- Custom cursor with smooth trailing animation
- GPU-accelerated 3D tilt effects (CSS transform: translateZ)
- Fully responsive across devices
- Dark/Light theme toggle
- Performance-optimized (no heavy libraries)

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## ⚡️ Performance Focus
Built without external animation libraries (like GSAP).  
All animations are handled using CSS transforms and lightweight JavaScript for optimal performance.

## 📂 Structure
- `index.html` - Main landing page  
- `projects.html` - Project archive page  
- `/projects` - Individual project pages  
- `/images` - Assets & previews  
- `script.js` - Animations & interactions  
- `style.css` - Styling & themes
