Mock Mate AI

Role

Frontend Engineer

Category

AI-Powered EdTech Tool

Tech Stack

React (Vite) Tailwind LLM API

Overview

MockMate AI is an AI-powered interview simulation platform designed to replicate real-world technical interviews with dynamic question generation and automated evaluation. Built with React and Vite, the app generates dynamic, role-specific questions and uses AI to evaluate responses against structured rubrics and provide real-time feedback.

The State Management Challenge

Ensuring the AI-generated evaluations were highly consistent, objective, and mapped perfectly to the React UI state during asynchronous evaluation loops proved incredibly difficult, as LLMs frequently hallucinate or break JSON formatting.

The Engineering Solution

I designed a structured prompt engineering pipeline with enforced JSON outputs to ensure reliable AI evaluations. On the frontend, I built a robust state management system using React Context to handle asynchronous evaluation flows, ensuring consistent UI updates even with delayed or malformed API responses.
