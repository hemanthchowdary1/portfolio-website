Penny Track

Role

Full Stack Engineer

Category

FinTech & AI Dashboard

Tech Stack

React.js Django REST Generative AI

Overview

PennyTrack is a smart, full-stack financial application that goes beyond standard expense tracking. By integrating a dynamic React frontend with a secure Django REST Framework backend, it provides users with interactive analytical charts and AI-driven insights into their financial habits.

The Data Challenge

The hardest challenge was generating meaningful, context-aware financial insights using Large Language Models without exposing sensitive user transaction data or hitting API rate limits during dashboard loads.

The Engineering Solution

I engineered a strict data-anonymization pipeline on the Django backend that aggregates and sanitizes spending patterns into general percentages before interfacing with the AI endpoint. To ensure lightning-fast dashboard renders, I implemented a robust caching layer for the AI responses, reducing external API calls by over 40% while still delivering highly personalized financial advice.
