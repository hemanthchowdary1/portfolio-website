Overview

The highly interactive personal portfolio website you are viewing right now is built entirely from scratch without frontend frameworks, serving as a live demonstration of hardware-accelerated native web development.

The Rendering Challenge

Implementing complex, mouse-tracking 3D tilt animations and custom cursors often causes severe layout thrashing, frame drops, and battery drain on mobile devices if relying on standard DOM manipulation.

The Engineering Solution

I avoided heavy JavaScript animation libraries (like GSAP) and instead built a lightweight custom Vanilla JS physics engine. By relying entirely on CSS transform: translateZ() for 3D depth calculations and wrapping event listeners safely, I pushed the rendering calculations directly to the GPU. I also implemented strict @media (pointer: fine) CSS guards to gracefully degrade the experience for touch devices, ensuring perfect optimization across all screen sizes.
