Post Craft

Role

Full Stack Developer

Category

Social / CMS Platform

Tech Stack

Python Django AJAX

Overview

PostCraft is a full-featured, secure blogging and social platform. It handles complex relational data, featuring robust user authentication, email OTP verification, an advanced notification system, and deep social interactions like following, liking, and bookmarking.

The UX Challenge

Traditional server-rendered Django applications often suffer from clunky page reloads during micro-interactions (like clicking a "heart" or "bookmark" button), which severely degrades the modern user experience.

The Engineering Solution

To create a Single Page Application (SPA) feel without the overhead of React, I developed a custom Vanilla JavaScript modular system utilizing the asynchronous fetch API. I implemented optimistic UI updates—meaning the UI instantly reflects likes and bookmarks to the user, while asynchronously syncing with the Django REST endpoints in the background. This resulted in a frictionless, highly responsive platform.
